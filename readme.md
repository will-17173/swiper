#vote


 > 基于swiper.js的焦点图, 选项卡, 组图组件

##how to demo

1. checkout
1. run `npm install`
1. run `spm install`
1. run `grunt`
1. view files in `/demo`

##how to use

1. run `spm install pandora/swiper`
1. write `require('pandora/mvote/VERSION.NUMBER/swiper')`

##find examples

1. view the source files in '/src'
